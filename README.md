# Vlog on Rails 8

MVP видеоблога на Ruby on Rails 8:
- создание поста с видеофайлом
- просмотр ленты постов
- просмотр отдельного видео
- комментарии к видео
- лайки и дизлайки к постам
- регистрация и вход пользователей

## Требования

- Ruby `3.4.8`
- SQLite3
- `ffprobe` (из пакета FFmpeg) для проверки кодека видео
- `yt-dlp` для импорта видео по ссылке YouTube

Поведение проверки кодека:
- `production`: строгая проверка кодека обязательна
- `development`/`test`: если `ffprobe` отсутствует, проверка кодека пропускается
- можно принудительно включить/выключить через `REQUIRE_FFPROBE=true|false`

Генерация превью:
- после создания поста запускается `GeneratePostThumbnailJob`
- задача извлекает 8 кадров на `10%`, `20%`, `30%`, `40%`, `50%`, `60%`, `70%`, `80%` длительности видео через `ffmpeg`
- кадр `30%` используется как основная `thumbnail`, остальные хранятся в `preview_frames`
- если `ffmpeg` отсутствует, пост будет доступен, но в ленте останется статус "Превью создается..."
- в `development` превью по умолчанию генерируется синхронно (без очереди), переключается через `THUMBNAIL_SYNC=true|false`

Автозаполнение при выборе видео:
- название берется из метаданных файла или из имени файла
- описание заполняется вручную

Импорт из YouTube:
- в форме создания поста можно вставить ссылку на YouTube и подтянуть описание
- кнопка `Скачать и опубликовать` показывает прогресс-бар скачивания на форме
- можно выбрать пресет качества `high` (до 1080p), `medium` (до 720p) или `low` (до 360p)
- сервер передает пресет в `yt-dlp`, предпочитая `AV1` (видео) и `Opus` (аудио), с fallback при недоступности
- скорость скачивания ограничена до `2M` (`--limit-rate 2M`)
- путь к бинарнику можно переопределить через `YT_DLP_PATH`

Приватность поста:
- при создании/редактировании поста можно выбрать видимость: `Все пользователи` или `Только я (приватный пост)`
- приватный пост виден только его автору

## Быстрый старт

```bash
bin/setup --skip-server
bin/rails server
```

Откройте `http://localhost:3000`.

## Проверка качества

```bash
bin/rails test
bin/rubocop
```

## Доступы

- публиковать видео могут только зарегистрированные пользователи
- редактировать и удалять пост может только его автор
- оставлять комментарии могут только зарегистрированные пользователи
- редактировать и удалять комментарий может только его автор
- ставить лайки и дизлайки могут только зарегистрированные пользователи
- неавторизованные посетители могут только просматривать посты

## Поддерживаемые видео

Поддерживаемые контейнеры:
- `video/mp4`
- `video/webm`
- `video/quicktime` (`.mov`)
- `video/x-matroska` (`.mkv`)

Поддерживаемые кодеки:
- `av1`
- `hevc` (H.265)
- `vp9`
- `h264`
